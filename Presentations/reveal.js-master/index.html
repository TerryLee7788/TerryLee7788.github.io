Grunt

The Javascript Task Runner

What is Grunt?

Grunt 是一個以 為基礎所開發的命令列工具
它可以將一些重複性的工作自動化, 例如精簡css code, livereload, 編譯CoffeeScript, LiveScript, less, sass...etc.

使用 下載以及安裝模組套件.

Getting Started

下載以及安裝
node.js.

install Grunt-cli


                            npm install -g grunt-cli

grunt-cli
只是負責執行Gruntfile 所設定的 Grunt 版本去執行指定的流程

2個重要的檔案

  1. package.json
  2. Gruntfile.js

package.json


                            npm init
                        

{
  "name": "yourname_here",
  "version": "1.0.0",
  "description": "Whatever you want",
  "devDependencies": {

  }
}

這個檔案是 npm 儲存一些資料的地方

devDependencies
用來讓你快速安裝所有npm 的module


                            npm install grunt --save-dev
                        

{
  "name": "yourname_here",
  "version": "1.0.0",
  "description": "Whatever you want",
  "devDependencies": {
    "grunt": "~0.4.0"
  }
}

Gruntfile.js

這個檔案通常包含下面幾個部分

Wrapper函數 


module.export = function (grunt) {
    // code start here
};

tasks設定


// 開始設置 tasks 物件
grunt.initConfig({
    uglify: {
      ...
    }
});

載入tasks


// 讀取 grunt tasks
grunt.loadNpmTasks('grunt-contrib-uglify');

預設tasks


// 預設 grunt default tasks
grunt.registerTask('default', ['uglify']);

新增Plugin 非常之簡單

Add sass plugin


npm install grunt-contrib-sass --save-dev
                        

// 開始設置 tasks 物件
grunt.initConfig({
    uglify: {
      ...
    },
    sass: {
      ...
    }
});

Add cssmin plugin


npm install grunt-contrib-cssmin --save-dev
                        

// 開始設置 tasks 物件

      ...
    },
    sass: {
      ...
    },
    cssmin: {
      ...
    }
});
